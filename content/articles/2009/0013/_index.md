# 侃用典的妙處

## 文／美国留学生

觀咸貓大神論述松鼠會的文字，頗有所感，做人文知識科普，確是實在且有意義的事情啊。我言語無味，自然不敢用“論”，“說”亦談不上，是以用“侃”，意在免責。

數落我掉書袋的義士自是不少，順便借此文給自己來點蒼白的辯護，以示我確然注意到了那些義士們的抗議，并無忽視他們且視其為蟲幺的嫌疑。

我所謂的用典，并非常人所詬病之踏襲傳統，咬文嚼字，賣弄玄虛；扭扭捏捏，裝腔做勢，甚至於神經過敏，出神見鬼。我所說的”典“既包括適逢其時的古典和不過是共黨極權下不便說話而用的”今典“，如草泥馬，俯臥撐，躲貓貓一類。用典的妙處，我個人歸結起來有三；其一曰情感的永恒；其二曰發見的意趣；其三曰低俗的掩飾。

所謂情感的永恒，其實最為淺顯。舉一通例以論之，唐人詩云：”秦時明月漢時關，萬里長征人未還。但使龍城飛將在，不教胡馬度陰山。”

起首一句“秦時明月漢時關”，便把整個戍邊守關的場景一下子推到了秦漢，“關，山，月”三個方塊字，亙古不變。而“龍城”，“飛將”，“陰山”一類的詞句，更為整個場景做了注腳。同時，這個場景本身便獲得了永恒。古往今來的邊塞詩文，都可以為這首詩做注解；而這首詩也可以作為那些詩文所表達的邊塞情義有力的旁證了。其中自是有卞之琳的橋上人之寓在，亦正是章學誠“六經注我，我注六經”的意味之所在了。真可謂“六經責我開生面”啦。

如果諸君還記得高三上學期語文課本中的那篇《說“木業”》的話，這個問題就更好理解了。但是基於我回憶起自己當時都對這篇東西感到厭煩的事實，還是和大家一起複習下吧。當然，我們這裡不探討木業村的前途問題，鳴人君和長門大神的未來只有AB他老人家的曉得。好了，說正題。林庚老先生提到屈原《九歌》寫道：“鳥鳥兮秋風，洞庭波兮木葉下。”

後世詩文便有了如謝莊《月賦》說：“洞庭始波，木葉微脫。”陸厥的《臨江王節士歌》又說：“木葉下，江波連，秋月照浦雲歇山。”至於王褒《渡河北》的名句：“秋風吹木葉，還似洞庭波。”則其所受的影響更是顯然了。更不必說“亭皋木葉下，隴首秋雲飛。”(柳惲《擣衣詩》

) 以及“九月寒砧催木葉，十年徵戍憶遼陽。”(沈佺期《古意》

) 根據林老先生的分析，“木業”常見的主要原因便是“自屈原開始把它準確地用在一個秋風葉落的季節之中，此後的詩人們無論謝莊、陸厥、柳惲、王褒、沈佺期、杜甫、黃庭堅，都以此在秋天的情景中取得鮮明的形象，這就不是偶然的了。”（但愿我沒背錯，不過大意應該就是這樣）所以，如果你要描述的那個場景正好與古人所見一致，用典本身就可以讓你不至於在“念天地之悠悠”之時，“獨愴然而涕下”了，因為前而可見古人，后而可待來者了嘛。不亦悅乎？

現在來談談所謂發見的意趣。以高中質檢現代文閱讀篇目《聽聽那冷雨》為例。我承認高中做那道題的時候，判定這無非又是余光中的另一篇統戰文章，與《鄉愁》同類而已罷了，套公式答題便是。我很感謝福州第一中學的郭蕙榕老師在評講那份練習的時候那樣悉心地跟我們講這篇文章，用了兩節課細細地品味分析，仿佛那是一篇重要的課文。那是高三下學期了，但是老師似乎并不在意。我自己當時愚鈍得很，沒能理解老師的用意，現在想起，實在是慚恨且遺憾啊。《聽聽那冷雨》開篇自然是以對大陸的遐思伊始，然後便轉入了當年（1974）的時事：“

安東尼奧尼的鏡頭搖過去，搖過去又搖過來。殘山剩水猶如是，皇天後土猶如是。紜紜黔首、紛紛黎民從北到南猶如是。那裏面是中國嗎？那裏面當然還是中國永遠是中國。只是杏花春雨已不再，牧童遙指已不再，劍門細雨渭城輕塵也都已不再。”，還有之後的“

在報紙的頭條標題裏嗎？還是香港的謠言裏？還是傅聰的黑鍵白鍵馬思聰的跳弓撥弦？還是安東尼奧尼的鏡底勒馬洲的望中？還是呢，故宮博物院的壁頭和玻璃櫃內，京戲的鑼鼓聲中太白和東坡的韻裏？”大家注意幾個關鍵詞：“安東尼奧尼”，“香港的謠言”，“傅聰”，“馬思聰”。當年安東尼奧尼拍的《中國》被匪共舉國批判，雖說根本沒人看過；香港左棍們的發癲搞得查良鏞先生寫點實在的社評都得收恐嚇信和炸藥包；更別說傅聰的“叛逃”，傅雷夫婦的上吊自殺以及馬思聰先生的遠走海外了。那個時代共黨把大陸搞得“率獸食人，人將相食”，確是顧亭林“亡天下”最好的詮釋了。基於此，余光中先生才發出了“那裡面是中國嗎？”的感嘆。這樣一想，也不難理解文章第一段的那句“

十五年，一切都斷了，只有氣候，只有氣象報告還牽連在一起，大寒流從那塊土地上彌天卷來，這種酷冷吾與古大陸分擔。”了，無非就是從西伯利亞來的蘇維埃的酷冷罷了。然後我們再來解讀文章最後的密碼：“一打少年聽雨，紅燭昏沉。再打中年聽雨，客舟中江闊雲低。三打白頭聽雨的僧廬下，這更是亡宋之痛，一顆敏感心靈的一生：樓上，江上，廟裏，用冷冷的雨珠子串成。”大家想想，何謂”亡宋之痛“啊？中華民國，亡國了嘛！大陸淪陷了嘛，而且轉眼便是淪陷一甲子了。

當然你自可以說我胡謅，跟劉心武一般胡鬧。但是不可否認如果你讀懂了其中所指代的意味，你所解開的密鑰便是你與作者的radical relationship了。既然是特別的access，則定然會有“花徑不曾緣客掃，蓬門今始為君開”的歡暢在了。胡蘭成云:"因為相知，所以懂得。“

張愛玲道：”因為懂得，所以慈悲“。雖不可同住同修，但是同緣同相，同見同知的意趣，便大可體會得到。

好吧，我承認先前列的第三條不過是調侃。只是想找藉口講個小故事供大家娛樂。話說數學大家陳省身先生頗好作詩，然則并不為佳。當然我并不是說數學家便做不好詩，比如蘇步青先生的七言絕句：“草草杯盤共飲歡，莫因柴米話心酸。春風已綠門前草，且耐余寒放眼看。”，簡簡單單，清清醒醒晰晰，有薄荷的香味在裡頭，用來下酒，自然酒味格外得好。糟了糟了，講故事離題了，回到陳先生身上。陳先生七十五歲壽辰時寫詩自賀曰：”百歲已過四分三“，為陳之藩先生所笑，稱其不過是犯了數學家的職業病，因為100*3/4=75嘛。然陳之藩先生的夫人童元方先生為省身先生辯護，說這其實是在用典，因為當年黃花崗七十二烈士之一的與校友林意洞齊名的趙聲志士，寫過”百年已過四分一，事業茫茫未可知“的句子。所以，陳省身先生職業病犯得有理有據有節！嘿嘿。

魏文帝曰：”文章者，經國之大業，不朽之盛事。“

放蕩為文如我者，看到這條箴言自然是要羞愧汗顏的。於是乎不敢再瞎寫了，江左版用典之三妙處到此結束。

謝謝觀賞。
